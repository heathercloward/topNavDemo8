DSP® Navigation and Menu Options

Browser Navigation

Page back, forward, and page refresh are achieved through browser navigation tools. Alternatively, the [F5] key can be used for page refresh. The user can also use the browser History option to view and link to previously accessed pages.

Site Toolbar Options

The Site Toolbar always displays at the top of the application. The toolbar contains, from left to right, the Site Link, Quick Links, Service Indicator, Design Menu, and Settings Menu.

Site Link

The Site Link icon redirects the user to the Site page, which is the default DSP® home page.

If the user has defined an alternative home page for their account, the link redirects the user to the defined page.

Clicking the down arrow expands the Site menu drop-down.

Quick Links

Quick links are bookmarks that allow users to link to frequently accessed pages. Quick links can be customized and can contain binding criteria.

NOTE: The Preferences link on the Settings menu opens a fly-out menu that allows the user to toggle the display of the Quick Links on the Site Toolbar.

Service Indicator

The color of the Service Indicator icon indicates the state of the CranSoftservice running in the background:

  • Green indicates the CranSoft service is running.

  • Yellow indicates one or more jobs have failed, but the CranSoft service is still running.

  • Red indicates the CranSoft service has stopped.

NOTE: To reset the Service Indicator to green, an Administrator must acknowledge all failed jobs in System Administration and the service must be running. The service can be managed from the Services console on the web server and can be configured to attempt to restart automatically after a failure.

Design Menu

The Design menu options depend on the functionality associated with the active page and the user's security settings.

Settings Menu

Click the Settings icon to expand the settings menu.

The Settings link opens the Settings page where users can change their personal settings, such as language, email address, password and telephone number.

The Preferences link opens a fly-out menu that allows the user to toggle the display of the Quick Links on the Site Toolbar.

The Help link opens a panel containing all available page level technical help, including page source, views, events and column property information.

The Logout link logs the user out of the current session and returns to the login page.

dspMigrate™ Context Menu

As a user works with components in dspMigrate™, the context (Wave Process Area) displays in the Context Bar (far left).

Selecting the context (Wave Process Area) opens a vertical fly-out that allows the user to change the context.

The menu options to the right of the context area allow the user to navigate to different components within the tool.

Refer to Context, the Context Bar, and Navigation in dspMigrate™ for more information.

Navigation Pane

The Navigation pane serves as the primary navigation for the DSP®. There are two navigation style options; Drill View and Tree View.

The options available on the Navigation pane depend on the users’ security access.

Tree View and Drill View

The Tree View and Drill View provide two options for navigating through pages in the DSP® .

 

Expand Menu Item

Select the name of the menu link to navigate to that page.

Sidebar Pin

The Sidebar Pin () icon opens or collapses the Navigation pane. The Navigation pane is pinned open by default.

When the Navigation pane is open, click the Sidebar Pin icon to collapse it. To reopen the Navigation pane click the Sidebar Pin icon or hover over the gray panel border that is visible when the Navigation pane is collapsed.

Quick Panel

The quick panel provides access to the Messages, Work List, Mapping Approval, Automation and dspCompose Requests panels. Click a Quick Panel icon to open the panel.

A user must be granted access to view the Mapping Approval tab. Refer to Approve or Reject Mappings in product help for more information.

If a user does not have access to dspCompose™ , the dspCompose Requests option does not display. Refer to Set up Security for dspCompose™ in product help.

The number on the icon indicates the number of outstanding items (messages, work list items or objects) related to the option.
